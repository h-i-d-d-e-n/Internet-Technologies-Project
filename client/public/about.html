About

"Endless Runner" is a browser-based sidescrolling game that we created as our project assignment. It combines web and game development into one cohesive product!

Team

This project was a group effort - Danielius and Dominykas worked together to create the site and the game. They focused on building the game itself, implementing functionality, and shaping the final design and feel of the project. Peculiar and Hassan J. worked on the report and presentation, while also assisting with the overall project whenever needed.

Tools List

  • We used JavaScript for the game and website logic.
  • We used PixiJS for rendering and game-related visuals as it's WebGL and uses JavaScript to display images and 2D content.
  • We used Vite for the project setup and our development workflow.
  • We used Piskel and Pixelartvillage for creating sprites and animations, as well as converting images to pixel art.

This website was created by Dominykas S., Danielius S., Peculiar O., and Hassan J.
