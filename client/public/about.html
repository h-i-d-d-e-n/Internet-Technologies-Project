About the Project

Endless Runner is a browser-based game project created as a team assignment. The project combined website development, game development, documentation, and presentation work into one complete final result.

Team Members

Danielius and Dominykas created the website and the game itself. Their work focused on building the game experience, implementing functionality, and shaping the final interactive design of the project.

The rest of the team, Peculiar and Hassan J., worked on the report and presentation, while also assisting with the overall project wherever needed. Their contribution helped complete the written, visual, and supporting parts of the assignment.

How the Project Was Made

The website and game were developed using modern web technologies and tools that supported both programming and visual asset creation.

  • PixiJS for rendering and game-related visuals
  • Vite for project setup and development workflow
  • JavaScript for the game and website logic
  • Piskel for creating sprites and animations
